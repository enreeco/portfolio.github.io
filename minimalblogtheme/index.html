Design Philosophy

A theme that feels like your code editor—quiet, focused, nothing shouting for attention.

Monochrome Palette

Black, white, and gray. No distractions, just clean typography and thoughtful spacing.

No Visual Clutter

No sliders, megamenus, or shortcodes that feel like they escaped from 2013. Just content.

Sharp Typography

Thoughtful spacing and readable fonts that make your words the star of the show.

Core Features

Responsive Grid Layout

CSS Grid with clamp-based typography ensures your content looks great on any device.

Three-Level Navigation

Multi-level menu support with clean, accessible navigation that works on mobile and desktop.

Custom Header & Logo

Full support for WordPress custom header images and site logos through the Customizer.

Styled Pagination

Beautiful post navigation cards and pagination with hover effects that spark joy.

Search Integration

Built-in search functionality with a clean, minimal search form design.

Comments Support

Fully integrated comments system with consistent styling that matches the theme.

Translation Ready

Built with internationalization in mind, ready for translation into any language.

Accessible HTML

Proper ARIA support, semantic markup, and keyboard navigation for all users.

Secure & Clean Code

Sanitized and escaped PHP everywhere. Built with security best practices in mind.

Installation

  1. 1

    Download the theme

    Get the zip file from the download link or clone the GitHub repository.

    Download ZIP
  2. 2

    Upload to WordPress

    Go to Appearance → Themes → Add New → Upload Theme and upload the zip file.

  3. 3

    Activate the theme

    Activate Minimal Blog by Enree.co from your themes list.

  4. 4

    Configure your site

    • Go to Appearance → Menus to define your main navigation menu
    • Visit Appearance → Customize → Site Identity for logo and Header Image for the banner
    • Configure Appearance → Widgets to set up Sidebar and Footer 1-4 widget areas
